# 邻选社区 - 社区私域购物平台

## 项目简介

基于微信小程序的社区私域购物平台，采用**团长+楼长双级分销**模式，为小区居民提供便捷的在线购物体验。

### 核心特色

- 🏘️ **小区绑定**：用户绑定所在小区和楼栋
- 👥 **双级分销**：团长（小区级）+ 楼长（楼栋级）两级分销
- 💰 **佣金分润**：楼长 5% + 团长 3% 佣金自动结算
- 🚀 **社区配送**：团长自提点 + 送货上门
- ⚡ **限时秒杀**：每日特价活动
- 💬 **邻里评价**：真实邻居评价和晒单

## 快速开始

### 环境要求

- Node.js >= 16
- npm >= 7
- 微信开发者工具

### 安装

```bash
cd mini-program
npm install
```

### 开发

```bash
# 微信小程序开发模式
npm run dev:weapp

# H5 开发模式
npm run dev:h5
```

### 构建

```bash
# 微信小程序构建
npm run build:weapp

# H5 构建
npm run build:h5
```

### 预览

1. 使用微信开发者工具打开 `mini-program/dist` 目录
2. 在 `project.config.json` 中替换你的小程序 AppID

## 分销模式说明

### 角色体系

```
团长（小区级）
  ├── 楼长A（1栋）
  │   ├── 用户1
  │   └── 用户2
  ├── 楼长B（2栋）
  │   └── 用户3
  └── 楼长C（3栋）
      ├── 用户4
      └── 用户5
```

### 佣金分配

| 角色 | 佣金比例 | 结算条件 |
|------|---------|----------|
| 楼长 | 5% | 用户确认收货后 T+7 |
| 团长 | 3% | 用户确认收货后 T+7 |

### 分销链路

1. 楼长生成专属邀请码/分享链接
2. 用户通过楼长链接进入小程序并下单
3. 系统自动记录分销关系
4. 订单完成后自动计算并结算佣金
5. 楼长可在分销中心查看收益并提现
