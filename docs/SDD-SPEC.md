## 1. 产品概述

### 1.1 产品定位
面向社区的私域流量购物平台，为小区居民提供便捷的在线购物体验，支持团长分销、社区配送等私域运营模式。

### 1.2 技术方案选择

| 维度 | 微信小程序 | 响应式Web | 原生APP |
|------|-----------|----------|---------|
| 开发成本 | 中 | 低 | 高 |
| 用户获取 | ★★★★★ | ★★★ | ★★ |
| 支付体验 | ★★★★★ | ★★★★ | ★★★★★ |
| 传播分享 | ★★★★★ | ★★★★ | ★★★ |
| 维护成本 | 中 | 低 | 高 |

**推荐方案：微信小程序（C端）+ Web管理后台（B端）**

### 1.3 核心用户角色
- **普通用户（C端）**：小区居民，浏览商品、下单购买
- **团长/分销员**：社区KOL，分享推广、获得佣金
- **管理员（B端）**：平台运营，管理商品/订单/用户
- **配送员**：负责最后一公里配送

## 2. 功能需求规格

### 2.1 C端小程序功能模块

#### F-001: 用户系统
- 微信一键授权登录
- 手机号绑定
- 收货地址管理（自动定位小区）
- 用户信息编辑

#### F-002: 首页与商品浏览
- Banner轮播广告
- 商品分类导航（图标式）
- 限时秒杀/拼团活动入口
- 商品瀑布流/列表展示
- 商品搜索（关键词搜索）
- 商品详情页（图片轮播、规格选择、评价展示）

#### F-003: 购物车
- 添加/删除/修改商品数量
- 商品选中/取消选中
- 价格实时计算
- 库存校验
- 空购物车状态

#### F-004: 订单系统
- 订单确认页（地址、商品、优惠券、配送方式）
- 多种支付方式（微信支付）
- 订单列表（按状态分类）
- 订单详情与物流追踪
- 订单评价
- 退款/售后申请

#### F-005: 营销系统
- 优惠券领取与使用
- 限时秒杀活动
- 社区拼团
- 团长分销（分享得佣金）
- 楼长分销（每栋楼独立分销员，团长下级）
- 新用户专享福利

#### F-008: 积分系统 ⭐ 新增
- **积分获取规则**：消费1元 = 1积分，订单确认收货后自动到账
- **积分永不过期**：所有积分永久有效，无失效时间
- **积分商城**：使用积分兑换社区服务
  - 🧹 保洁服务（如：1000积分/次）
  - 🚗 洗车服务（如：800积分/次）
  - 🏠 免物业费（如：30000积分/月）
  - 更多服务可扩展
- **积分明细**：积分获取/消费记录，支持分页查询
- **积分展示**：首页、个人中心显眼位置展示当前积分
- **下单提示**：订单确认页预估可获得积分

#### F-009: 配送系统
- 团长自提点取货
- 楼长/团长送货上门
- 配送方式自由选择
- 配送状态追踪

#### F-006: 社区功能
- 小区绑定与切换
- 团长/分销员展示
- 社区公告/活动通知
- 邻里评价/晒单

#### F-007: 消息通知
- 订单状态推送
- 促销活动通知
- 物流更新提醒
- 系统消息

```sql
-- 用户表
users: id, openid, nickname, avatar, phone, community_id, role, created_at

-- 商品分类表
categories: id, name, parent_id, icon, sort_order

-- 积分表
user_points: id, user_id, total_points, created_at, updated_at

-- 积分流水表
points_records: id, user_id, order_id, type(earn/spend), amount, 
                description, service_type, created_at

-- 积分服务表（可兑换服务）
points_services: id, name, icon, points_price, market_price, 
                 category(cleaning/car_wash/property_fee/other), 
                 stock, status, description, created_at
```

### 3.4 API接口规范

遵循 RESTful 风格：

```
GET    /api/v1/products              # 商品列表
GET    /api/v1/products/:id          # 商品详情
POST   /api/v1/orders                # 创建订单
GET    /api/v1/orders                # 订单列表
GET    /api/v1/orders/:id            # 订单详情
PUT    /api/v1/orders/:id/status     # 更新订单状态
POST   /api/v1/cart/items            # 添加购物车
DELETE /api/v1/cart/items/:id        # 删除购物车商品
POST   /api/v1/payment/unified-order # 微信支付统一下单

# 积分相关
GET    /api/v1/points/info           # 积分总览
GET    /api/v1/points/records        # 积分流水
GET    /api/v1/points/services       # 可兑换服务列表
POST   /api/v1/points/exchange       # 积分兑换服务
GET    /api/v1/points/exchanges      # 兑换记录
```
